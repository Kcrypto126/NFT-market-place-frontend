### NFT Marketplace Frontend (Production)

## Techstack

- Next.js v14 (React 18)
- Ethers.js v6
- Tailwind CSS with Shadcn/UI
- React Hook Form

## To be determined

- Redux/Zustand/MobX (State Management)
- Custodial Wallet
- React Query v5

## How to run

1. Install node modules

```
npm i
```

2. Run development environment

```
npm run dev
```
